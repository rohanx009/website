## ğŸš€ How to Play

### Web Version (Recommended)
1. Open `chess_game.html` in any modern web browser
2. Click on a piece to select it (must be your turn)
3. Valid moves will be highlighted in **gold**
4. Click on a highlighted square to move
5. Captured pieces appear in the side panels

```bash

```
- Enter moves in format: `e2 e4`
- Type `help` for all commands
- Type `quit` to exit
